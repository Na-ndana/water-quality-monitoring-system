# IoT-Based Smart Water Quality Monitoring and Borewell Management System

## Overview

This project presents a low-cost IoT-based groundwater quality monitoring system using Arduino Uno and multiple environmental sensors.

The system continuously monitors water quality parameters and classifies water as SAFE or UNSAFE based on BIS and WHO standards.

## Features

- Real-time Water Quality Monitoring
- pH Measurement
- TDS Monitoring
- Turbidity Detection
- Temperature Monitoring
- Ultrasonic Water Level Detection
- Finite State Machine (FSM) Based Control

## Hardware Components

- Arduino Uno
- pH Sensor / Potentiometer
- TDS Sensor
- Turbidity Sensor
- DS18B20 Temperature Sensor
- Ultrasonic Sensor
- LEDs and Buzzer

## Technologies Used

- Arduino IDE
- MQTT Protocol
- IoT
- Embedded Systems
- Finite State Machine (FSM)

## Applications

- Groundwater Monitoring
- Borewell Monitoring
- Smart Agriculture
- Environmental Monitoring
- Rural Water Management
